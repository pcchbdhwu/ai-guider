# 第一个 PR 实操指南

## 推荐练习仓库
访问：https://github.com/firstcontributions/first-contributions
这是一个专门让新手练习 PR 流程的仓库。

## 完整步骤

### 1. Fork 仓库
在 first-contributions 仓库页面右上角点击 **Fork** 按钮 → 在你的账号下创建副本。

### 2. Clone 到本地
```bash
git clone https://github.com/<你的用户名>/first-contributions.git
cd first-contributions
```

### 3. 创建分支
```bash
git checkout -b add-your-name
```
分支名随意，这里用 `add-your-name` 作为例子。

### 4. 添加你的名字
在 Contributors.md 文件末尾添加一行你的名字，保存。

### 5. 提交
```bash
git add Contributors.md
git commit -m "docs: 添加彭晨晟到贡献者列表"
```

### 6. 推送
```bash
git push -u origin add-your-name
```

### 7. 创建 PR
1. 打开你 Fork 后的仓库页面
2. 点击 "Compare & pull request" 按钮
3. 填写 PR 标题和描述
4. 点击 "Create pull request"
5. 等待维护者合并！

## 验证标准

- [ ] 成功 Clone 到本地
- [ ] 成功创建新分支
- [ ] 成功 Commit 并 Push
