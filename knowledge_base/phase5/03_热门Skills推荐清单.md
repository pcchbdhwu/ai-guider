# 热门 Skills 推荐清单

## 官方 Skills（来自 anthropics/skills）
| Skill | 用途 | 触发场景 |
|-------|------|---------|
| **skill-creator** | 创建自定义 Skill | 用户想创建新 Skill |
| **security-review** | 代码安全审查 | 检查安全漏洞 |
| **review** | Pull Request Review | 审查代码变更 |
| **simplify** | 代码质量优化 | 重构、简化代码 |
| **find-skills** | 搜索和安装 Skills | 查找新工具 |
| **init** | 初始化 CLAUDE.md | 新项目开始 |
| **loop** | 定时循环任务 | 周期性检查 |
| **claude-api** | Claude API 开发 | 开发 API 应用 |
| **cherry-assistant-guide** | Cherry Studio 使用指南 | Cherry Studio 问题 |

## 安装方式
1. 对话中自然触发（Agent 自动识别）
2. `/skill-name` 显式调用
3. `mcp__skills__skills` action="install" 安装
