# Cherry Studio Agent 与 MCP 初体验

## 什么是 Agent（智能体）
在 Cherry Studio 中，Agent = 系统提示词 + 模型 + 知识库 + MCP 工具 + 全局记忆 的组合体。

你可以创建多个 Agent，分别用于不同的场景。比如：
- 一个专门帮你写论文的 Agent
- 一个专门帮你学编程的 Agent
- 一个专门帮你翻译的 Agent

## Agent 创建步骤（Cherry Studio 中）
1. 左侧菜单栏 → 点击「Agent」标签
2. 点击「新建 Agent」按钮
3. 填写配置：
   - **名称**：给 Agent 起个名字（如"AI启蒙导师"）
   - **描述**：简要说明 Agent 的用途
   - **系统提示词**：这是核心！告诉 Agent 它是什么角色、怎么说话、能做什么
   - **模型**：选择 Agent 使用哪个模型
   - **知识库**：选择 Agent 可以查阅的知识库文件
   - **MCP 工具**：勾选 Agent 可以调用的外部工具
4. 保存后即可在对话中切换使用

## 系统提示词写什么
系统提示词 = Agent 的"人设说明书"，包含：
1. **角色**：你是什么专家？
2. **能力范围**：你能做什么？
3. **行为准则**：你怎么说话、不能做什么？
4. **输出格式**：你用什么格式回答问题？

## MCP 是什么
MCP（Model Context Protocol）= AI 连接外部工具的标准协议。

举个例子：AI 本身不能搜索网页，但通过 MCP 连接了搜索工具后，AI 就能在需要时自动调用搜索。

## 在 Cherry Studio 中配置 MCP
1. 设置 → MCP 服务器 → 添加服务器
2. 填写服务器名称和地址
3. 启用后，Agent 即可调用该服务器提供的工具

## 全局记忆
Cherry Studio 的全局记忆功能让 Agent 记住跨对话的重要信息。比如：
- "我叫彭晨晟，是电子信息专业大二学生"
- "我习惯用 DeepSeek 做日常问答，Claude 做代码分析"

设置路径：设置 → 全局记忆
