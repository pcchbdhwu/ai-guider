## 选模型决策树

```
你需要做什么？
├── 写代码/调试Bug → Claude > DeepSeek > GPT
├── 写中文文章/翻译 → DeepSeek > 文心一言 > 通义千问
├── 分析图片/截图 → Gemini 2.5 Pro(免费) > GPT-5.4 > Claude > 豆包(免费) 
├── 日常问答/学习 → DeepSeek(便宜) > 豆包(免费) > 通义千问
├── 长文档分析(>100页) → Kimi > Claude > 通义千问
├── PPT/Word/Excel办公 → 通义千问 > WPS AI > Gamma
├── 创意写作/剧本 → GPT-5.4 > Claude > DeepSeek
├── 预算极有限 → DeepSeek(最便宜付费) > 豆包(免费) > Gemini(免费额度)
└── 多模型协作(规划+编码+验收) → GPT做规划 + Claude做编码 + 专门Agent做验收
```

## 新手推荐配置（Cherry Studio 三件套）

1. **日常主力**：DeepSeek-V3（性价比之王，日常学习编程足够）
2. **深度任务**：Claude Sonnet 4.6（复杂代码/长文分析时切过去）
3. **图片/多模态**：Gemini 2.5 Pro（免费额度大，需要图片分析时用）或豆包（完全免费备用）

## 价格常识

以 100万 Token 为单位（约等于写一本《三体》第一部的文字量）：
- DeepSeek：约 ¥1-4 元
- Claude Sonnet：约 $15 元（≈¥105）
- GPT-5.4：约 $15-75 元（≈¥105-525）

学生建议：日常用 DeepSeek，关键时刻用 Claude，图片分析用 Gemini/豆包。
