# AI 视频创作入门

## AI 视频生成完整流程

```
脚本构思 → 画面生成 → 配音配乐 → 剪辑合成 → 导出成品
   ↓           ↓          ↓          ↓
DeepSeek    可灵/Sora  剪映TTS    剪映/CapCut
Claude      即梦         elevenlabs  Premiere
```

## Step 1：用 AI 写视频脚本与分镜

### 脚本 Prompt 模板
> "你是一个短视频导演。我要做一个关于[主题]的短视频，时长约[60]秒，发布在[抖音/B站/视频号]，受众是[大学生]。请输出：(1)完整口播文案 (2)每句话对应的画面描述 (3)建议的BGM风格"

### 分镜 Prompt 模板
> "请将上述脚本拆解为分镜表，包含：镜号/时长/画面描述/运镜方式/字幕文案/备注"
