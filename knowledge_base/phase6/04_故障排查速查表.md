# 工具链故障排查速查表

## Claude Code
| 症状 | 可能原因 | 解决 |
|------|---------|------|
| command not found | 未安装/PATH问题 | 重装：npm install -g @anthropic-ai/claude-code |
| 认证失败 | Key错误/过期 | 检查环境变量，Console重建Key |
| 限流 | 请求过频 | 等待1-2分钟，检查多实例 |
| 额度不足 | 余额用完 | Console充值或等重置 |
| 权限拒绝 | 安全机制 | 弹窗中Allow或/config调整 |
| DeepSeek 404 "env" | 版本更新兼容模式变化 | settings.json 添加 `"CLAUDE_CODE_SIMPLE": "1"` |

## DeepSeek API
| 症状 | 可能原因 | 解决 |
|------|---------|------|
| 官方繁忙 | 高峰期限流 | 切换到Azure/火山引擎备选渠道 |
| 返回截断 | maxTokens不够 | 增大maxTokens参数 |
| 中文混乱 | Temperature太高 | 降低到0.3-0.5 |

## Git/GitHub
| 症状 | 可能原因 | 解决 |
|------|---------|------|
| Permission denied | SSH Key未配置 | 检查~/.ssh配置，添加到GitHub |
| Merge conflict | 分支冲突 | git status看冲突文件，手动解决后commit |
| push被拒 | 远程有新提交 | git pull --rebase 后再 push |

## Cherry Studio
| 症状 | 可能原因 | 解决 |
|------|---------|------|
| 模型连接失败 | API Key错误或网络问题 | 测试连接，检查代理设置 |
| MCP工具不显示 | MCP Server未启动 | 检查MCP配置，重启Cherry Studio |
| 知识库检索不准 | 文档未正确分块 | 重新上传，调整分块大小 |

## 通用排障流程
1. 复制完整报错信息
2. 粘贴到 AI 或搜索引擎
3. 检查 GitHub Issues 是否有人遇到相同问题
4. 重启相关服务再试
