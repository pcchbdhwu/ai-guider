# AI 工具链全景图

## 六层工具架构
```
┌─────────────────────────────────────────────────┐
│ 对话层    │ Cherry Studio（统一入口）              │
│ 模型层    │ Claude/DeepSeek/GPT/Gemini/豆包        │
│ 代码层    │ Claude Code CLI（AI辅助编程）           │
│ 协作层    │ GitHub（版本控制+开源协作）             │
│ 扩展层    │ Skills + MCP（能力拓展生态）            │
│ 办公层    │ WPS AI / Gamma / 剪映（文档/PPT/视频）  │
└─────────────────────────────────────────────────┘
```

## 工具-场景映射矩阵
| 学习场景 | 工具选择 |
|---------|---------|
| 日常AI问答/学习 | Cherry Studio + DeepSeek |
| 复杂编程任务 | Claude Code + Claude |
| 图片/报错截图分析 | Cherry Studio + Gemini/豆包 |
| 中文写作/论文 | Cherry Studio + DeepSeek |
| 版本控制/开源 | GitHub + Git CLI |
| PPT/文档制作 | DeepSeek生成内容 + WPS排版 |
| 视频创作 | DeepSeek(脚本) + 即梦(画面) + 剪映(剪辑) |
| 知识库+RAG | Cherry Studio 知识库功能 |
| Agent扩展 | Skills 市场 + 自定义 Skill |
| 多模型协作 | GPT规划 + Claude编码 + Agent验收 |
