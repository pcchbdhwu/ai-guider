# Claude Code 安装完整指南

## Windows (WSL2 方案，推荐)
```bash
# 1. 安装 WSL2（Windows 功能中开启"适用于 Linux 的 Windows 子系统"）
wsl --install

# 2. 进入 WSL 终端，安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# 3. 验证 Node.js
node --version  # 应 >= 18.0.0
npm --version

# 4. 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 5. 验证安装
claude --version
```

## Windows (PowerShell 直接方案)
```powershell
# 1. 下载安装 Node.js Windows 版
# 访问 https://nodejs.org/ → 下载 LTS 版本 → 安装

# 2. 打开 PowerShell，验证
node --version
npm --version

# 3. 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 4. 验证
claude --version
```

## Mac
```bash
# 安装 Homebrew（如已安装跳过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 验证
claude --version
```

## 首次启动
```bash
# 在任意目录启动
claude

# 或指定工作目录
claude /path/to/your/project
```

首次启动会引导你输入 Anthropic API Key 完成认证。
