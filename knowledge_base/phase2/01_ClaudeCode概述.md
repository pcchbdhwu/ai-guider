# Claude Code 概述

## 什么是 Claude Code
Claude Code 是 Anthropic 公司推出的终端 AI 编程助手。它运行在你的命令行终端中，可以：
- 阅读和理解你整个项目的代码
- 执行终端命令（创建文件、运行测试、安装依赖）
- 操作 Git 版本控制（提交、分支、PR）
- 通过自然语言完成复杂的编程任务

## 和网页版 Claude 的核心区别
| 维度 | 网页版 Claude | Claude Code |
|------|-------------|------------|
| 运行位置 | 浏览器 | 终端 |
| 代码能力 | 只能看粘贴的代码 | 直接读取整个项目 |
| 执行能力 | 不能执行命令 | 可以运行命令和脚本 |
| Git | 不能操作 | 直接操作 Git |
| 上下文 | 单次对话 | 理解整个代码库结构 |
| 适用场景 | 对话、写作、分析 | 编程、项目开发、自动化 |

## 类比理解
- 网页版 Claude = 一个坐在你旁边的编程高手，你只能把代码截图给他看
- Claude Code = 一个能直接操作你电脑的编程高手，能自己看项目、写代码、跑测试

## 适用场景
- 新项目从零搭建
- Bug 定位与修复
- 代码重构
- 自动化脚本编写
- 代码审查
- 学习新语言/框架

## 系统要求
- Node.js 18+ 环境
- Windows 建议通过 WSL2 使用（体验更好）或直接 PowerShell
- Mac/Linux 原生支持
- 需要 Anthropic API Key
