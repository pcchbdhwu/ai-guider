# Claude Code 核心命令速查表

| 命令 | 作用 | 使用场景 |
|------|------|---------|
| `/help` | 查看所有可用命令 | 忘记命令时 |
| `/clear` | 清空当前对话上下文 | 开始全新任务 |
| `/compact` | 压缩对话上下文（保留关键信息） | 对话太长导致响应变慢/成本变高时 |
| `/config` | 打开配置文件 | 修改模型、权限、主题等设置 |
| `/cost` | 查看当前对话的 Token 消耗与花费 | 关注预算 |
| `/add-dir` | 添加工作目录 | 让 Claude Code "看到"更多文件夹 |
| `/init` | 初始化项目 CLAUDE.md 文件 | 新项目开始时，告诉 Claude Code 项目背景 |
| `/review` | 审查当前分支的代码变更 | PR 提交前自查 |
| `/security-review` | 安全审查当前变更 | 检查是否有安全漏洞 |
| `/simplify` | 优化代码质量 | 重构后检查 |
| `/vim` | 切换 Vim 编辑模式 | Vim 用户 |
| `/status` | 查看当前会话状态 | 了解当前上下文大小、模型等 |
| `/logout` | 退出登录 | 切换账号 |
| `/doctor` | 诊断环境问题 | 安装或运行出问题时 |

## IDE 协议命令（在 IDE 中使用时）
| 命令 | 作用 |
|------|------|
| `/ide` | 查看 IDE 集成状态 |
| `/terminal-setup` | 配置终端集成 |

## 快捷操作
- `Ctrl+C`：中断当前生成
- `Ctrl+D`：退出 Claude Code
- `↑/↓`：浏览历史命令
- `!` 前缀：以 Bash 模式执行命令（如 `!git status`）
