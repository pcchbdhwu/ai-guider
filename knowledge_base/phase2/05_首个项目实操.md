# Claude Code 首个项目实操

## 项目：用 Claude Code 写一个 Python 计算器

### Step 1：创建项目目录
```bash
mkdir my-calculator
cd my-calculator
```

### Step 2：启动 Claude Code
```bash
claude
```

### Step 3：向 Claude Code 描述需求
```
帮我用 Python 写一个命令行计算器，要求：
1. 支持加减乘除四则运算
2. 能处理用户输入的错误（如输入字母）
3. 输入'q'退出程序
4. 每步运算后显示结果，允许继续运算
5. 代码加注释
```

### Step 4：观察 Claude Code 自动工作
Claude Code 会：
1. 分析需求，规划代码结构
2. 创建 calculator.py 文件并写入代码
3. 可能要求你确认是否执行某些操作（根据权限设置）

### Step 5：测试运行
```bash
python calculator.py
```
测试各种输入：正常计算、除以0、输入字母、退出。

### Step 6：如果有 Bug，让 Claude Code 修复
```
我测试发现当输入空字符串时程序崩溃了，请修复这个问题。
```

### Step 7：让 Claude Code 添加功能
```
请添加以下功能：
1. 支持幂运算（^）
2. 显示计算历史记录
3. 添加清空历史的命令
```

## 关键要点
- **描述需求要具体**：明确输入输出、边界条件、错误处理
- **观察 Claude Code 的工作方式**：它会自动浏览项目文件、写代码、运行测试
- **迭代改进**：第一次生成的代码不完美很正常，继续提要求让 Claude Code 改
- **注意权限弹窗**：Claude Code 执行文件操作时会请求确认，这是安全机制
