# Claude Code 常见报错解决

## 报错1：`claude: command not found`
**原因**：Claude Code 未正确安装或 PATH 未包含 npm 全局包路径

**解决**：
```bash
# 检查 npm 全局包路径
npm list -g --depth=0

# 重新安装
npm install -g @anthropic-ai/claude-code

# 检查 PATH（Windows PowerShell）
# 将 npm 全局路径添加到系统 PATH
```

## 报错2：`Authentication failed / Invalid API Key`
**原因**：API Key 错误、过期或未设置

**解决**：
1. 确认 Key 完整复制（以 `sk-ant-` 开头）
2. 在 console.anthropic.com 检查 Key 状态
3. 尝试创建新 Key
4. 检查环境变量：`echo $ANTHROPIC_API_KEY`

## 报错3：`Rate limit exceeded`
**原因**：短时间内请求过多次，触发限流

**解决**：
1. 等待 1-2 分钟后重试
2. 检查是否开启了多个 Claude Code 实例
3. 升级 API 套餐提升限流额度

## 报错4：`Insufficient quota`
**原因**：账户余额/额度不足

**解决**：
1. 登录 console.anthropic.com 查看余额
2. 充值或等待下月额度重置
3. 考虑切换模型（Sonnet→Haiku 更便宜）

## 报错5：`Permission denied`（执行命令被拒）
**原因**：Claude Code 的权限设置阻止了该操作

**解决**：
1. 在权限弹窗中选择"Allow"
2. 或在 `/config` 中调整权限策略
3. 理解这是安全机制，不是 Bug

## 报错6：Windows WSL 相关报错
**原因**：WSL 未正确安装或版本过低

**解决**：
```bash
# 升级 WSL
wsl --update

# 检查 WSL 版本（需要 WSL2）
wsl --version

# 重启 WSL
wsl --shutdown
```

## 报错7：`context window exceeded`
**原因**：对话上下文超过了模型的上下文窗口

**解决**：
1. 使用 `/compact` 压缩上下文
2. 使用 `/clear` 开始新对话
3. 将长任务拆分成多个短对话

## 报错8：`API error 404 "env"`（DeepSeek 接入时）
**原因**：Claude Code 更新后，新版对第三方模型兼容模式发生变化（2026年6月）

**解决**：
在 `~/.claude/settings.json` 中添加顶层字段：
```json
{
  "CLAUDE_CODE_SIMPLE": "1"
}
```
注意：这是顶层字段，与 `mcpServers`、`models` 同级。添加后重启 Claude Code 即可。

## 通用排障流程
1. 先看报错信息 → 复制到搜索引擎/问 AI
2. 运行 `claude --doctor` 诊断环境
3. 检查 GitHub Issues：https://github.com/anthropics/claude-code/issues
